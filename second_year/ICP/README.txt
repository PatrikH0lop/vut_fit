# CBlockEditor
ICP Project - Block editor

Tool for designing, editing and saving block schemes.
Example scheme created by authors might be found in folder 'examples'.
Tutorial for usage is integrated in the application, open it by choosing About -> About | Tutorial in the application menu.

# How to run
Compile it with 'make' in root folder.
To compile and run the application use 'make run' command.
